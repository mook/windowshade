WindowShade

WindowShade is a quick-and-dirty import of the old Gonzo miniplayer to Songbird 1.4 (which has a much larger miniplayer by default).

Screenshot

Download

Distribution is on the Songbird add-ons site.

Source Code

Source code is available on GitHub. GPL 2, due to the use of files from the original Gonzo.

Feather Support

Please see this guide to add WindowShade support to your custom feather.

Contributing

If you find any problems, feel free to put messages on the Songbird add-ons site, or contact me on IRC (server: irc.mozilla.org, channel: #songbird, nick: Mook or Mook_sb). I'm usually also reachable over email, if you can find an address. Forking on GitHub works too, of course.

Naming

WinAMP 2 had a windowshade mode. This is that, so that the player can (more or less) fit into the title bar area of windows.
